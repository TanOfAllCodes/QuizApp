# QuizApp

A simple quiz application designed to help educators understand how children learn.

[Click here to try the QuizApp!](https://tanofallcodes.github.io/QuizApp/)

## Overview

QuizApp is a child-friendly quiz platform with a clean, neutral, and light-colored aesthetic. It supports English and German, offers 10 questions with randomized answer options, and provides immediate feedback with explanations. Users can navigate freely, view results, and restart or end the quiz at any time.

### Quiz Interface
The quiz consists of 10 questions, each with four answer options displayed in random order, with only one correct answer. The interface is intuitive, with navigation controls for a seamless experience.

- **First and Last Questions**: The first question hides the "Previous" button, and the last question hides the "Next" button for logical navigation.

- **Navigation**: Users can navigate freely using "Next" and "Previous" buttons or the top "Question Navigation" boxes. The current question is highlighted with a thick border, colored green (correct) or red (incorrect) if answered.

- **Answer Confirmation**: The "Confirm Answer" button is clickable only when an answer is selected on an unanswered question.

### Question Feedback
After answering, immediate feedback is provided:
- Correct answers highlight the selected option in green.
- Incorrect answers mark the selected option in red and highlight the correct answer in green.
- All questions include an explanation for the answer.
